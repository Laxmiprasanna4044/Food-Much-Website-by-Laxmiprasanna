# Food Munch Website

## 📌 Project Overview

Food Munch is a responsive food ordering website built using HTML, CSS, and Bootstrap. It provides an attractive and user-friendly interface to explore delicious food items, offers, and services.

## 🚀 Features

* Responsive navigation bar
* Banner section
* Why Choose Us section
* Explore Menu section
* Delivery & Payment section
* Thank You section
* Follow Us section
* Font Awesome icons
* Responsive design using Bootstrap

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 4
* Font Awesome

## 📂 Project Structure

* `index.html` – Main webpage
* `style.css` – Stylesheet
* `images/` – Project images

## 🎯 Learning Outcome

This project helped me improve my skills in HTML, CSS, Bootstrap, responsive web design, and creating modern user interfaces.
